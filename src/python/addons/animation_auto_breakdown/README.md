# Auto Breakdown 自動中割り
## Auto Breakdown And Relational Functions
## 自動中割り及び関連機能

## 必要環境
* Blender Version 2.79 が動作する環境

## 開発環境  
* Blender Version 2.79
* CPU    Intel Core i5
* Memory 16 GB


## インストール方法
  BlenderのUser Preferences の Add-onsタブより
  本アドオンを有効にしてください。

## アンインストール方法
  BlenderのUser Preferences の Add-onsタブより
  本アドオンを無効にしてください。
